# javaNavent

## Stack utilizado:

- Java 1.8
- Spring Boot Framework
- Gradle
- Bootstrap 4
- jQuery
- sweetAlert2

## Ejecución:

En la raiz del projecto se agregó el jar ejecutable de la aplicación.
Para ejecutarlo, ubicarse en la raiz del projecto e ingresar el siguiente comando.

$ java -jar TestNavent.jar
